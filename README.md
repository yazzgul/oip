## Задание 4 (директория task04): Вычисление TF-IDF

### Описание
Скрипт на Python вычисляет метрику TF-IDF для всего корпуса документов из `task01`, используя токены и леммы из `task02`. Для каждого документа отдельно считается TF-IDF по точным формам слов (термины) и по приведённым к начальной форме (леммы).

- **TF** (term frequency) — доля вхождений слова в данном документе относительно общего числа слов
- **IDF** (inverse document frequency) — `log(N / df)`, где N — число документов, df — в скольких документах встречается слово
- **TF-IDF** = TF × IDF

**Цель задания**: реализовать вычисление статистических весов терминов и лемм для последующего ранжирования в поисковой системе.

### Требования к окружению
- Python 3.7 или выше
- Дополнительных библиотек не требуется

### Установка и запуск

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/ваш-username/ваш-репозиторий.git
   cd ваш-репозиторий/task04
   ```

Результаты появятся в папке `results/`:
   ```
   results/
     terms/
       page_0001_terms.txt
       page_0002_terms.txt
       ...
     lemmas/
       page_0001_lemmas.txt
       page_0002_lemmas.txt
       ...
   ```

### Формат выходных файлов

Каждая строка: `<слово> <idf> <tf-idf>`

```
автомобиль  0.693147  0.000412
беспилотник 1.098612  0.000823
разработчик 0.405465  0.000206
```
